## M8.5 notes

**Traffic KPIs are unchanged by design** — M8.5 made zero traffic-sim changes (the
fourth milestone running), so every signal/yield/grid/roundabout/gradesep metric should
read identical to M8 (±0% band). That identity is itself the headline check: covered
tunnels and trenches are invisible to routing (an under-crossing is already
grade-separated), so nothing in the sim moved.

**Perf pass (the queued M8.5 profiling task).** Profiling the fuzz slowdown that
negative elevation introduced located the draw-time cost in `RoadNetwork.Validate`'s two
per-edge scans — the crossing-intersection loop and `OverlapsExisting`'s 33×E
`ClosestPoint` sweep — neither of which had a spatial prefilter (the M8 invariant's
`CheckEdgeCrossings` already had one). Adding the same convex-hull AABB + Y-band reject:

| measurement | before | after |
|---|---|---|
| `perf.validate500_ms` (long flat diagonal, 480-edge grid) | 59.1 ms | **10.4 ms** (−82%) |
| `perf.validate500_graded_ms` (same, +20 m — grade-separated) | — | **9.8 ms** (new; skips the whole grid) |
| fuzz 1500×3 wall-clock | 5m24s | **1m25s** |

The prefilters are behaviour-preserving: an XZ-disjoint edge cannot intersect the
proposal (Bézier convex-hull property), and an edge whose Y-range is ≥ `MinClearance`
from the proposal's yields only grade-separated crossings, which the loop already
discards — so a prefiltered edge is exactly the work the body would have thrown away.
The 385-test quick gate and the 3×10k fuzz (which drives Validate on every draw)
confirm identical outcomes.

Secondary: `RoadEdge.MinRadius`/`MaxGradient` are now lazily cached (the curve is
immutable; every edit builds a fresh instance), giving ~1.8× on the isolated
`NetworkInvariants.Check` sweep. A crossing-pair memo explored during profiling was
**dropped** — it pinned dead edges through the run and regressed the dynamic fuzz loop;
the finding is recorded in the commit message so it isn't re-attempted.

The fuzz alphabet's off-ground draws now split 15% up / 15% down (matching the pre-M8.5
30% off-ground frequency, only the sign is new) plus a `ToggleCovered` action, so
certification wall-clock stayed tractable while exercising trenches and tunnels.
