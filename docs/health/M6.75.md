## Notes (M6.75, editor-only milestone)

- Every traffic metric is bit-identical to the M6.5 baseline — the milestone touched
  only the editor (snapping, ghost rendering, audio), never the sim. Deltas of +0,0%
  are exact, not rounded.
- **perf.validate500_ms 142 ms (+140% vs baseline 59 ms) is environmental, not a code
  regression**: an A/B worktree run of the pre-M6.75 commit (`2db3294`) on the same
  machine, same day measured 134 ms. The metric is a single unrepeated Stopwatch
  sample of one `Validate` call and tracks machine state; the hard 150 ms ceiling —
  the actual gate — holds. Worth re-baselining on a quiet machine next milestone.
- Certification fuzz: 3 seeds × 10,000 actions green (exit 0, 13 min run), with
  `SnapTypes.CellLength` newly in the toggle alphabet.
